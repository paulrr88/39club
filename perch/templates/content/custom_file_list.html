Instructions:

Each file on the policies list has its own distinct page on Perch that you can use to edit it (with this being one of them). You will see a list of these pages when you first click on the "Policies files" link in order to edit the list (you likely came here from that page).

On each list item page, you can either:
- Edit an existing file by changing the fields that are currently on this page
- Add a new file to the list by scrolling to the bottom of this page and clicking "Save & Add another", which will result in a new page loading containing a new blank set of fields (added to the bottom of the current policies files list) that can then be filled in as below with appropriate text and a link to a file. See below for instructions:

- "Text to display in link to file" field: this is what the link will say, for instance "2017 report document", which would then output a link saying just that, which the user can then click to be taken to the download in question.
- "Download type" field: this is the file type in question which is displayed after the link to tell the user what it is that they are downloading. For instance, PDF or Word Document. You can also cheat and use this field to add any other information related to the file if you want, just be careful to not make the explanation too long or it might distort the page layout!
- "File" field: this area is where you can select your file. To do this you will need to:
- Click "Select or upload a file"
- In the resulting Select an Asset area that pops up, either:
- Click an existing item that you want to link to and then click "Use Selected", or
- Click the "+Add Asset" button in the top right, and then either drop a file into the indicated box area or click the box area to select the file from your computer.
- Click the file that you just added, now in a little box in the "Select an Asset" area
- Click "Use Selected" in the top right of the "Select an Asset" section ("Use Selected" should be highlighted green if you've correctly clicked a file to be used, if not just try clicking again or "Use Selected" won't do anything)
- N.B. if you enter this area by accident and want to go back, click the grey (X) icon in the far top right of the screen, or press ESC on your keyboard.
- PDF format for documents is preferable if you can manage it as more people can read this format than items created in Word etc! And of course PDFs can't be edited.

To delete an item: click the little red "Delete" button to the top right of one of the sections below.

To reorder the policies list: go to the "Policies files" page (mentioned at the top here) so you can see the full list of all the items in the policies files list. On the top right should be a button labelled "Reorder", click this and it will change the appearance of the list, and you can now click and drag each item to reorder it. Click the "Save Changes" button at the bottom of the list when finished.
